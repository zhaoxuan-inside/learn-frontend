# CSS 选择器，为满足条件的元素设置样式

> CSS 选择器和 HTML 选择器正好相反 [HTML 选择器](./_000_html_selector.md)

## CSS 选择器语法格式

### 1. 元素包含指定属性的元素的样式

```css
[元素类型][[属性]] {
    [样式]
}
```

### 2. 属性为指定属性值的元素的样式

```css
[元素类型][[属性]="[属性值]"] {
    [样式]
}
```

### 3. 属性值中包含指定内容的元素的样式

#### 3.1. 属性值包含指定单词内容的元素的样式(分词方式为空格)

```css
[元素类型][[属性]~="[属性值]"] {
    [样式]
}
```

#### 3.2. 属性值只要包含指定内容即可

```css
[元素类型][[属性]*="[属性值]"] {
    [样式]
}
```

### 4. 属性值以指定内容开头的元素的样式

#### 4.1. 以指定值开头，且后面一个字符若存在则只能是连字符 |= 

```css
[元素类型][[属性]|="[属性值]"] {
    [样式]
}
```

#### 4.2. 以指定值开头即可 ^=

```css
[元素类型][[属性]~="[属性值]"] {
    [样式]
}
```

### 5. 属性值以指定内容结束的元素的样式

```css
[元素类型][[属性]$="[属性值]"] {
    [样式]
}
```

## 优先级和层叠

当两个规则同时匹配同一个元素时，后声明的规则会覆盖前一个层叠顺序决定
