# 优先级
## 1. 优先级计算基础（四元组）
1. CSS 优先级用一个四元组表示
    A. 内联样式
    B. ID 选择器
    C. 类、伪类、属性选择器
    D. 元素、伪元素选择器
2. 四元组从左到右逐级比较，数值大的优先级高；全相同则后出现的覆盖先出现的。
