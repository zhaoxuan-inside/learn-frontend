# 正则表达式
用来在文本中搜索数据

## 语法

### Reg 定义
const reg = new RagExp('[pattern]', '[flags]');
const reg = '/[pattern]/[flags]'

### flag 
1. d 
    - 生成索引。在匹配结果中返回每个捕获子串的起始和结束索引位置
2. g
    - 全局匹配。查找所有匹配项，而不是在第一个匹配后停止
3. i
    - 忽略大小写。匹配时，不区分大小写字母
4. m
    - 多行匹配。让^和$可以匹配每行的开头和结尾，而不仅是整个字符串的开头和结尾
5. s
    - dotAll 模式。允许英文句号.匹配包括换行符\n在内的任意单个字符。
6. u
    - Unicode 模式。将模式视为 Unicode 代码点序列，能正确处理四字节的 UTF-16 字符。
7. v
    - Unicode 集合模式。是u标志的升级版，支持在字符类中使用集合运算（如交集&&、差集--）。
8. y
    - 粘性匹配。从正则表达式对象的lastIndex属性指定的位置开始匹配，且必须匹配成功，否则会失败。

### 1. String 对象方法
1. [str].match([reg])
    - 匹配结果数组或 null
2. [str].matchAll([reg])
    - 所有匹配迭代器 (需 g 标志)
3. [str].search([reg])
    - 返回第一个匹配的索引
4. [str].replace([reg], [new_str])
    - 替换匹配的内容
5. [str].split([reg])
    - 用正则分割字符串

### 2. RagExp 对象方法
1. [reg].test([str])
    - 检测字符串是否匹配，结果为 true|false
2. [reg].exec([str])
    - 匹配结果数组或 null

## 用途
1. 搜索
2. 替换
3. 验证
4. 提取
