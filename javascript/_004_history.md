# history 对象
history 对象是浏览器提供的一个**内置对象**
代表了**当前窗口**的**浏览历史记录**（即当前标签页中访问过的页面列表）
通过 history 对象，可以编程式地控制浏览器的**前进**、**后退**，以及**修改历史记录条目**

# 基本属性

## history.length
返回当前标签页的历史记录栈中的条目数量
⭐ **注意**：包括当前页面

## history.state
返回当前历史记录条目的状态对象

# 常用方法

## history.back()
加载历史记录中的**前一个页面**（相当于点击浏览器的"后退"按钮）

## history.forward()
加载历史记录中的**下一个页面**（相当于"前进"按钮）

## history.go(delta)
加载相对于当前页面的某个历史页面。

delta 为整数：
- 正数表示前进
- 负数表示后退。

例如：
go(0) 相当于刷新当前页面

# 修改历史记录的方法

## history.pushState(state, title, url)
**新增**一个新的条目到历史记录，并**改变**当前地址栏的 URL，**但不会加载新页面**

## history.replaceState(state, title, url)
**替换**历史记录中的一个条目，**不刷新页面**

## 参数说明

### state
任意 JavaScript 对象，与该历史条目关联
可通过 `history.state` 或在 `popstate` 事件中获取

### title
大多数浏览器忽略此参数，通常传空字符串 ''

### url
新的 URL（相对于当前页）
**不能跨域**，否则会抛出异常

## 应用
`pushState` 和 `replaceState` 不会触发事件，一般通过手动劫持这两个方法，被调用时执行自己的逻辑
