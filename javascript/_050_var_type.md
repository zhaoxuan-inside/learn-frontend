# 变量的定义
1. var var_name
    - 全局的变量
2. let var_name
    - 只作用在确定的作用于
3. const var_name
    - 常量

# 变量类型
## 1. 基本类型
- 1.1. undefined
    - 未定义的
- 1.2. number
    - 数值
- 1.3. boolean
    - 布尔类型
- 1.4. string
    - 字符串
- 1.5. function
    - 函数
- 1.6. symbol
    - 符号类型
- 1.7. bigint
    - 大整数

## 2. 对象类型
- 2.1. object
    - 对象
- 2.2. Date
    - 时间类型
- 2.3. array
    - 列表类型
💫 注意：这三种类型的对象，通过 typeof 方法查看到都是 object 类型；

## 3. 没有任何值的类型
- 3.1. null
- 3.2. undefined
    - 未定义的
💫 注意： null == undefined 为 true ； null === undefined 为 false；也就是这两个对象值相等，但是类型不一样；
